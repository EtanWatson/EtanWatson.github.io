Spring Boot 概述

『习惯优于配置』

Spring Boot核心功能

  • 可以以jar包的形式独立运行 java -jar xx.jar
  • 内嵌Servlet容器，无需以war包形式部署项目
  • 提供starter简化Maven配置
  • 自动配置Spring
  • 准生产的应用监控 Spring Boot提供基于http、ssh、telnet对运行是的项目进行监控
  • 无生成代码和xml配置：Spring Boot不需要任何xml配置即可实现Spring的所有配置

Spring Boot的优缺点

优点：

  • 快速构建项目
  • 对主流开发框架的无配置集成
  • 项目可独立运行，无需外部依赖Servlet容器
  • 提供运行是的应用监控
  • 极大的提高了开发、部署效率
  • 与云计算天然集成
